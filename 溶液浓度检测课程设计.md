# 光电信息科学与工程 - 溶液浓度检测课程设计

## 一、设计目的

通过建立基于光学吸收原理的溶液浓度检测系统，使学生掌握：
1. **光学基本原理** - 朗伯-比尔定律的应用
2. **光电传感技术** - 光电二极管、光电倍增管等传感器的使用
3. **信号处理方法** - 模拟信号的采集与数字处理
4. **系统设计能力** - 从光学设计到电子电路再到软件算法的完整流程

## 二、理论基础

### 2.1 朗伯-比尔定律
```
A = εbc = log₁₀(I₀/I)
```
其中：
- **A** - 吸光度（无量纲）
- **ε** - 摩尔吸收系数（L·mol⁻¹·cm⁻¹）
- **b** - 光程长度（cm）
- **c** - 溶液浓度（mol/L）
- **I₀** - 入射光强度
- **I** - 透射光强度

### 2.2 光电检测原理
- 光电二极管将光信号转换为电流信号
- 光电倍增管用于弱光检测（灵敏度高）
- 输出信号与光强成正比

## 三、系统设计方案

### 3.1 光学系统设计

#### 3.1.1 硬件组成
```
光源 → 准直光学系统 → 被测溶液 → 集光系统 → 光电探测器
         ↓
    参考光路（暗场）
```

**主要光学元件：**
- **单色光源或LED**：选择特定波长（推荐：500nm-700nm范围）
- **准直透镜**：确保平行光入射
- **比色皿**：标准光程长度通常为1cm
- **集光透镜**：提高光收集效率
- **光电二极管/PMT**：作为探测器

#### 3.1.2 光学参数设计

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 光源波长 | 600nm | 根据溶液吸收特性选择 |
| 光源功率 | 5-100mW | 确保信噪比 |
| 准直光束直径 | 3-5mm | 完全通过比色皿 |
| 光程长度 | 1cm | 标准规格 |
| 探测器孔径 | 相当于入射光束 | 最大化集光效率 |

### 3.2 电子电路系统

#### 3.2.1 前置放大电路
```
光电探测器 → 跨阻放大器(TIA) → 反演放大器 → 低通滤波器 → 模数转换器
```

**关键指标：**
- **跨阻放大系数**：10⁶ - 10⁹ Ω（根据光电流大小调节）
- **噪声等效功率(NEP)**：< 10⁻¹⁵ W/√Hz
- **带宽**：DC-1kHz（足以满足静态检测需求）

#### 3.2.2 信号调理电路

1. **AC耦合 vs DC耦合**
   - DC耦合：适合低频缓慢变化信号
   - AC耦合：消除零点漂移

2. **滤波器设计**
   - 类型：巴特沃斯低通滤波器
   - 截止频率：1kHz
   - 阶数：2-4阶

#### 3.2.3 参考电路设计
```
      R
   ----
   |    |
  (D)  |OP| ----o--- V_ref
   |    |
  GND  |
   
用参考光路消除光源强度波动
```

### 3.3 数据采集与处理系统

#### 3.3.1 硬件平台
- **单片机**：STM32系列 / Arduino
- **ADC参数**：12位或更高分辨率，采样率 > 10kHz
- **通信**：USB或串口与PC通信

#### 3.3.2 软件算法流程
```
初始化系统
    ↓
零点校准（测量暗场）
    ↓
参考光校准（测量参考液）
    ↓
采集多个数据点（N > 100）
    ↓
数据滤波（移动平均/卡尔曼滤波）
    ↓
计算透射比 T = I/I₀
    ↓
计算吸光度 A = log₁₀(1/T)
    ↓
浓度计算 c = A/(εb)
    ↓
显示结果 + 不确定度分析
```

#### 3.3.3 关键算法实现

**移动平均滤波（降低噪声）：**
```python
def moving_average(data, window_size=10):
    filtered = []
    for i in range(len(data) - window_size + 1):
        filtered.append(sum(data[i:i+window_size]) / window_size)
    return filtered
```

**吸光度计算：**
```python
def calculate_concentration(V_sample, V_ref, epsilon, path_length):
    """
    V_sample: 样品测量电压
    V_ref: 参考光电压
    epsilon: 摩尔吸收系数
    path_length: 光程长度(cm)
    """
    T = V_sample / V_ref  # 透射比
    A = math.log10(1/T)   # 吸光度
    c = A / (epsilon * path_length)  # 浓度
    return c
```

## 四、实验步骤

### 4.1 系统搭建与调试

1. **光学系统搭建**
   - 搭建光路，确保准直光束质量
   - 用功率计验证光强分布
   - 调整光束通过比色皿的位置和角度

2. **电路调试**
   - 检查电源极性和接地
   - 用示波器测量各节点信号
   - 调整跨阻放大器增益

3. **系统集成**
   - 烧录单片机程序
   - 建立通信接口
   - 运行自检程序

### 4.2 标定实验

**步骤：**
1. 配制一系列浓度已知的标准溶液（5-10个浓度点）
   - 例如：0 mol/L, 0.5, 1.0, 1.5, 2.0, 2.5 mol/L
   
2. 对每个浓度进行10次独立测量

3. 绘制标定曲线：吸光度 vs 浓度
   - 应为线性关系（R² > 0.99）

4. 用线性回归拟合得到斜率和截距

### 4.3 性能测试

| 指标 | 测试方法 | 验收标准 |
|------|---------|---------|
| 测量范围 | 用不同浓度溶液测试 | 0-5 mol/L |
| 准确度 | 与分析仪比对 | 偏差 < 5% |
| 精密度 | 同一样品10次测量 | 相对标准偏差 < 3% |
| 重复性 | 不同时间测量 | 结果相同 |
| 响应时间 | 测定稳定所需时间 | < 10s |
| 线性范围 | 吸光度不超过0.1-2.0 | R² > 0.99 |

### 4.4 误差分析

**主要误差来源：**

1. **光学误差**
   - 光源强度波动：±2% → 用参考光补偿
   - 光束散射：设计准直系统
   - 偏振效应：选择非偏振光源

2. **电子噪声**
   - 热噪声：FN = 4kTBR
   - 1/f噪声：降频采样时明显
   - 量化噪声：增加ADC位数

3. **化学因素**
   - 温度效应：Δc/c ≈ 0.3%/°C → 恒温
   - pH影响：控制在±0.1pH范围
   - 溶液浑浊：<0.001浊度单位

4. **人为操作误差**
   - 比色皿不洁：定期清洗
   - 气泡影响：排除气泡
   - 校准不准：多次重复校准

**不确定度合成：**
```
U总² = U读数² + U标定² + U温度² + U其他²
相对不确定度 = U总/测定值
```

## 十、附录：程序示例（Arduino/C语言）

```cpp
#include <Wire.h>
#include <math.h>

// 引脚定义
const int SENSOR_PIN = A0;
const int LED_PIN = 3;

// 传感器参数
const float EPSILON = 10.0;      // 摩尔吸收系数 L·mol⁻¹·cm⁻¹
const float PATH_LENGTH = 1.0;   // 光程长度 cm
const float V_REF = 1023.0;      // 参考电压（8位ADC最大值）

// 数据缓冲
const int BUFFER_SIZE = 100;
int sensorBuffer[BUFFER_SIZE];
int bufferIndex = 0;

void setup() {
  Serial.begin(9600);
  pinMode(LED_PIN, OUTPUT);
  digitalWrite(LED_PIN, HIGH);   // 打开LED
  delay(1000);  // 等待LED稳定
  Serial.println("光学浓度检测系统已启动");
  calibrate();  // 系统标定
}

void loop() {
  // 采集样品信号
  float concentration = measureConcentration();
  
  // 显示结果
  Serial.print("浓度(mol/L): ");
  Serial.println(concentration, 4);
  
  delay(2000);  // 2秒采集一次
}

// 系统标定函数
void calibrate() {
  Serial.println("\n开始系统标定...");
  Serial.println("请确保液体是参考液（或蒸馏水），然后按任意键继续");
  
  while(!Serial.available());
  while(Serial.available()) Serial.read();
  
  float sumV = 0;
  for(int i = 0; i < 50; i++) {
    sumV += analogRead(SENSOR_PIN);
    delay(10);
  }
  V_REF = sumV / 50.0;
  
  Serial.print("参考电压: ");
  Serial.println(V_REF);
  Serial.println("标定完成！\n");
}

// 浓度测量函数
float measureConcentration() {
  float sumV = 0;
  
  // 采集10个数据点
  for(int i = 0; i < 10; i++) {
    int rawValue = analogRead(SENSOR_PIN);
    sumV += rawValue;
    delay(100);
  }
  
  float avgV = sumV / 10.0;
  
  // 计算透射比
  float T = avgV / V_REF;
  T = constrain(T, 0.001, 0.999);  // 防止log(0)或log负数
  
  // 计算吸光度
  float A = -log10(T);
  
  // 计算浓度
  float c = A / (EPSILON * PATH_LENGTH);
  
  return c;
}
```
